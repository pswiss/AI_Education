# AI Educational Resources
Funded by an award from the NJIT Grace Hopper Research Institute
to Professors Dr. Petras Swissler and Dr. Sangwoo Park

---

### Description
This is the github repository that holds the outputs of the project funded by the NJIT Grace Hopper Research Institute. This project proposed to explore the intersection of swarm robotics and power systems, as well as to generate a robust set of educational materials to aid future educators in developing lessons about the ever-more-important field of artificial intelligence.
These outputs can be divided into three general categories, listed below

### Research Papers
The folder ```research_papers``` contains the manuscripts for the papers that came out of this project. This folder will be updated as new papers are accepted for publication, and supplemental material such as videos and datasets will also be hosted here, as permitted by copyright agreements.

### AI Concept Visualizations
The folder ```concept_visualizations``` contains various interactive visualizations of AI concepts, including

- K-means Clustering
- Neural Networks
- Gradient Descent

### AI Summer Camp
In the summer of 2025 Drs. Swissler and Park hosted a summer camp at the New Jersey Institute of Technology. The folder ```ai_summercamp``` contains the lecture slides, project materials, as well as an instructor guide for the summer camp. The hope is that this can serve as a turnkey solution to anyone wishing to run a similar summer camp at their own school.
