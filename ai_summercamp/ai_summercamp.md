# AI Summer Camp

---

This folder contains the material used for the summer camp run at NJIT in the summer of 2025
There are four files in this folder

- ai_summercamp.md: You've found it!
- Camp_Program.zip: This contains all the material used for the summer camp, including lecture slides, assignment templates, and completed code
- Summer_camp_manual.pdf: This contains a detailed overview of all the material for the summer camp. It is essentially a guide for people wanting to run a similar camp in the future.
- survey_questions.xlsx: These are questions for intake and exit surveys that will help you understand the impact of the summer camp.
